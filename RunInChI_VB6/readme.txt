RunINCHI_VB6  contains a MS Visual Basic 6 example.
============  It has a simple Mol/SDfile reader and shows how to call
              InChI library dll from a MS Visual Basic 6 application.

              In case of problems displaying RunINCHI window please
              read the following:

RunINCHI Window size and position settings are stored
in the Windows Registry under the key
HKEY_CURRENT_USER\Software\VB and VBA Program Settings\RunINChI\Settings
In case of problems displaying RunINCHI main window you may try
to delete them and rerun RunINCHI.exe.
